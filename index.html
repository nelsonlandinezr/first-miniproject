CSS Shippet Cheatsheet

You and your group can decide which CSS styles and colors you will use to design the application, but the app needs to be a responsive. Use the following images to gain an understanding of how the app should look at different screen sizes, from a layout perspective

Flexbox Row

Use these three properties to create a Flexbox row layout

.row{

display: flex;

flex-direction: row;

flex-wrap: wrap,

}

Flexbox Column

Use these three properties to create a Flexbox column layout

.column{

display: flex;

flex-direction: column;

}

CSS Grid Layout

Build 12-column layout using Css Grid

.grid{

display: grid;

width:100%;

grid-template-column:

repeat(12,1fr)

}
