Flexbox Row

Use these three properties to create a Flexbox row layout

.row{
    display: flex;
    flex-direction: row;
    flex-wrap: wrap,
}

Flexbox Column

Use these to create a Flexbox column layout

.column{
    display: flex;
    flex-direction: column;
}

CSS Grid Layout

Build 12-column layout using CSS Grid

.grid{
    display: grid;
    width:100%;
    grid-template-column:
    repeat(12,1fr);
 }

Linear Gradients

This will create a Background linear gradient from top to bottom

.linear-gradient-bacground{
    background-image: linear-;
    gradient(
        rgba(232,102,236,0.3) 0%,
        rgba(232,102,236,0.6) 100%
        );
}

    transition: all 0.5s easy-in-out;
}

.code-card:hover .card-header{
